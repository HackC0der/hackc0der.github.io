二、skills推荐

anthropics/skills

anthropics/skills 是 Anthropic 官方开源的 Claude 技能系统实现仓库，核心为 Claude 大模型提供可动态加载的模块化能力扩展方案，让 Claude 能按任务需求加载专业指令、脚本和资源，成为特定领域的专属专家，该仓库也是 Agent Skills 开放标准的核心实践载体Anthropic。

或者拷贝所有技能：

  1. 把 skills 文件夹 copy 到 ~/.claude/skills/ 或项目 .claude/skills/
  2. Claude Code 会自动扫描并加载。
  3. 使用：在对话里直接提技能名或由claude自行选择

ECC：everything-claude-code

参考：Anthropic 黑客松冠军项目 Everything Claude Code 完整上手攻略 - 知乎

Everything Claude Code 是由 Anthropic 黑客松冠军 Affaan Mustafa 创建的开源项目，提供了一套较为完整的 Claude Code 配置体系。其核心思路不是把 Claude Code 当作单一的编程工具，而是将其配置成一个由不同角色组成的「虚拟开发团队」。

Planning with Files

Claude Code 自带的 Plan Mode 有个让人头疼的问题：规划存在对话上下文里，上下文一压缩就丢了。长任务做到一半，Claude 忘了自己在干嘛。又从头开始。

Planning with Files 把规划、进度和知识都写进 Markdown 文件。Claude 开始干活前先创建计划文件，每完成一步就更新进度，遇到有用的信息就记到知识文件里。文件在磁盘上就不会丢，即使上下文被压缩了也能恢复状态。
