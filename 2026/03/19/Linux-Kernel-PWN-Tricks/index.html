一、常用工具

image解包脚本

经典的 Linux 内核工具 extract-vmlinux，核心作用是从压缩的 Linux 内核镜像中提取出未压缩的 vmlinux（内核可执行文件）。

这个脚本的目标是：自动识别内核镜像的压缩格式，解压并提取出完整的、可识别的 ELF 格式 vmlinux 文件。

它支持识别并解压多种压缩格式的内核镜像（gzip/xz/bzip2/lzma/lzop/lz4/zstd），最终输出纯净的 vmlinux。

  1. 核心功能：从压缩的 Linux 内核镜像中自动识别压缩格式、解压并提取出 ELF 格式的 vmlinux 文件；
  2. 关键逻辑：通过 “魔数定位压缩头 → 解压 → ELF 校验” 的流程，依次尝试多种主流压缩格式；
  3. 使用场景：内核开发 / 调试中，需要获取未压缩的 vmlinux 用于反汇编、调试、分析内核符号等场景。

vmlinx-to-elf

vmlinux-to-elf 的作用就是：

  1. 解析原始 vmlinux 镜像的结构（支持 ARM/ARM64/x86/x86_64 等主流架构）；
  2. 恢复 / 提取符号表、段信息、调试信息；
  3. 输出一个标准的、可被调试器识别的 ELF 文件。
